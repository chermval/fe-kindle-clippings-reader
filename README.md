# kindleClippingsReader
Proyecto que permite leer los archivos `My Clippings.txt` de Kindle donde se guardan las notas, subrayados y marca paginas en el dispositivo. 

### Utiliza:
- node v14.13.0
- npm v6.15.8
- react v17.0.1
- bootstrap v4.5.3
- axios v0.21.0
